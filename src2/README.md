# 기능목록

- [/] 게임시작 문구 출력
- [/] 다리 길이 입력 기능
- [/] 다리 길이에 따른 다리 생성
- [/] 이동할 칸을 선택하는기능
  - [/] 윗칸은 U
  - [/] 아래킷은 D
- [/]입력한 칸에 따른 다리 생성 기능
  - [/] 갈수 있으면 O
  - [/] 갈 수 없으면 X
  - [/] 선택하지 않은 칸은 빈칸
- [ ] 이동한 다리를출력하는 기능.
- [ ] 다리 이동 이후 결과를 판단하는 기능
  - [ ] 실패시 다시 시작하는 기능
    - [ ] 카운트 1증가하는 기능
    - [ ] 다리를 초기화 하는 기능
  - [ ] 성공시 다시 시작하는 기능
- [ ] 성공 또는 실패에 따른 텍스트를 출력하는 기능.
