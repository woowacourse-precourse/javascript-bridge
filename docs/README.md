다리 게임 기능 목록
=================
* 다리 길이 입력 받기
    * 예외
    * 다리 랜덤 생성 
* 이동할 칸 입력 받기
    * 예외
    * 이동 후 다리 상황 출력
* 실패 시 재시작/종료 입력 받기
    * 예외
    * 재시작 시 처음 생성한 다리로 재시작
    * 종료 시 최종 결과 출력
* 최종 결과 출력
    * 다리 최종 결과 출력
    * 성공 여부 출력
    * 시도 횟수 출력


예외 처리 목록
=========
### 다리 길이 입력
* 3 이상 ~ 20 이하의 숫자가 아닐 시
### 이동할 칸 입력
* U (위) 또는 D (아래) 문자가 아닐 시 
### 게임 재시작/종료 입력
* R (재시작) 또는 Q (종료) 문자가 아닐 시
