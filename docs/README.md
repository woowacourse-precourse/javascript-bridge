#기능 요구 사항

다리 건너기 게임 -위아래를 선택해 한칸만 건널 수 있는 다리를 끝까지 건너가는 게임이다.

##입력
[] 자동으로 생성할 다리 길이를 입력 받음. (3~20 숫자만)
[] 입력값이 올바르지 않다면 예외 처리함.
[] 게임 재시작/종료 입력 받음.
[] R(재시작)/Q(종료) 중 하나의 문자를 입력 받음. 입력값이 올바르지 않다면 예외처리함.
-------------------------------------------------------------------------------------
##출력
[] 게임 시작 문구를 출력함.
[] 다리의 형태는 [] 로 표시함.
[] 다리 칸의 구분은 | 문자열로 구분함.
[] 이동할 수 있는 칸을 선택한 경우에는 O표시, 이동할 수 없는 칸을 선택한 경우에는 X표시함.
[] 선택하지 않은 칸은 공백 한칸으로 표시함.
[] 현재까지 건넌 다리를 모두 출력함.
[] 예외 상황 시 에러 문구를 출력함. "[ERROR]"로 시작함
-------------------------------------------------------------------------------------
##구현 계획
[] 다리 길이를 입력받음.
[] 다리의 길이를 입력받으면 다리의 형태[]에서 다리 길이만큼 공백설정함.
[] 다리 칸의 구분은 다리 길의-1 개의 문자열 |로 구성됨.
[] 다리를 만들었다면, 위(U),아래(D)의 선택에 따라 이동할 수 있는 칸인지 확인한다.
[] 이동할 수 있다면 O, 없다면 X를 표시한다.
[] X가 선택되었다면, 게임재시작(R)/게임종료(Q)를 선택지를 제공함.
[] 선택지에 따라 게임을 재시작, 게임을 종료시킴.
[] 게임을 종료했다면, 게임 실패를 출력하고 지금까지의 게임 시도 횟수를 출력함.
[] O가 선택되었다면, 다음 칸으로 이동해 U,D 선택에 따라 다리 끝에 도착할때까지 위의 절차를 반복함.
[] 다리 끝에 도착했다면, 게임 성공을 출력함.
[] 다리 끝에 도착할때까지의 게임 시도 횟수를 출력함.
