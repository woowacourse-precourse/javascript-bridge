# 기능목록

1. - [ x ] 다리길이를 입력받는다. - 입력받은 길이로 다리 생성
      - 예외
        - 3이상 20이하만 입력받는다.
        - 숫자만 입력받는다.
        - 1개만 입력받는다.


2. - [ ] 이동할 칸을 입력받는다.
   - 예외
     - U,D만 입력받는다.
     - 소문자를 입력하면 오류를 발생시킨다.
     - 한개만 입력받는다.


3. - [ ] 이동한 칸이 O 이면 [ O ] 표시를 하고 다음 칸을 입력받는다.
   - 맞추면 move로 다시 이동값을 확인하고 다음 칸 물어보기
   - 틀리면 retry로 다시 할건지 확인하고 다음 게임 여부 물어보기


4. - [ ] 최종게임 결과 출력하기
   - 결과 맵 출력하기
   - 게임 성공여부 출력하기
   - 총 시도 횟수 출력하기
