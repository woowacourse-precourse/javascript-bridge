## 🕹 실행 Flow

1. 게임 시작 출력
2. 다리 길이 입력 받아 게임 반복 횟수 설정
3. 이동할 칸 선택 받기 (U/D)
4. 랜덤으로 건너갈 수 있는 다리를 설정하여 판단
5. 맞으면, 끝날 때까지 반복을 진행한다
6. 틀리면, 게임을 끝낸다
7. 재시작(R)하면, 시도 횟수를 늘리고 게임을 진행한다
8. 종료(Q)하면 게임을 끝 출력
