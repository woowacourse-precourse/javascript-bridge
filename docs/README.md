## 기능 목록

- 게임 시작 문구 출력 기능

- 다리 길이 입력 기능

  - 입력값 유효성 검사 기능

    - 3 이상 20 이하의 숫자

    - [예외사항] 숫자가 아닌 경우

    - [예외사항] 3 미만의 수

    - [예외사항] 20 초과의 수

  - 다리 생성 기능

- 이동 칸 입력 기능

  - 입력값 유효성 검사 기능

    - U 혹은 D

    - [예외사항] 소문자 u 혹은 d

    - [예외사항] U 혹은 D가 아닌 문자

  - 이동 성공 여부 확인 기능

- 이동 지도 출력 기능

  - 이동 지도 생성 기능

- 재시작 여부 입력 기능

  - 입력값 유효성 검사 기능

    - Q 혹은 R

    - [예외사항] 소문자 q 혹은 r

    - [예외사항] Q 혹은 R이 아닌 문자

  - 게임 재시작 기능

  - 게임 종료 기능

- 게임 종료 문구 출력 기능

  - 게임 성공 여부 확인 기능

  - 총 시도한 횟수 계산 기능
