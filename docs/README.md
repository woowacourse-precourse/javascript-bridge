# validator
| 기능 | 예외사항 |
|-|-|
| 다리 길이 검사 | 입력값이 숫자가 아닌 경우 <br/> 입력값이 자연수가 아닌 경우 <br/> 입력값이 3 이상 20 이하가 아닌 경우 |
| 이동할 칸 검사 | `U` 또는 `D` 이외의 모든 입력 |
| 재시도 여부 검사 | `R` 또는 `Q` 이외의 모든 입력 |
