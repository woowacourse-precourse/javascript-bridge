## 구현 설계

1. 다리 길이 입력

- 3 이상 20 이하
- 3 미만 20 초과: 예외 처리 '[ERROR]' 다리 길이는 3부터 20 사이의 숫자여야 합니다.

2. 이동 방향 입력

- U: 위로 이동
- D: 아래로 이동
- 이외: 예외 처리 '[ERROR] 잘못된 방향입니다.'

3. 다리 클래스

- 입력 받은 길이의 다리 생성
- 각 다리의 칸은 0,1 중 무작위로 값을 이용하여 위 아래 방향 지정
- 게임이 재시작되어도 다리 인스턴스는 삭제하지 않고 그대로 사용

4. 예외 처리

- 예외 발생 후 해당 지점부터 다시 입력을 받음
